Web Development
===============
-Developing a website

1. FrontEnd : User Interface (Look & Feel)
    HTML : Content ( paragraph, div, images, table ,form, audio , video)
    CSS :  Styling ( border, color, width,height, placement)
    Bootstrap : CSS Framework (pre-defined CSS classes)
    Javascript : Behaviour
    Angular/React/Vuejs: Javascript framework (Rapid Application Development)

2. BackEnd : Develop REST APIs (Java, .net, python , PHP, Nodejs)

3. Database : Holds the data ( Oracle , MySql , mongoDB )
